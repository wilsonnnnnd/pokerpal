# Security Reporting

Thank you for caring about PokerPal security. If you find a security issue, please follow these steps:

1. Do not open a public issue with sensitive details.
2. Send a private email to: hdpokerxone@gmail.com.
3. Provide:
   - A short summary of the issue.
   - Steps to reproduce.
   - Affected versions or commits.
   - Any PoC code or screenshots (avoid leaking user data).

We will acknowledge your report within 3 business days and share remediation steps.

Responsible disclosure: do not exploit the issue; give us reasonable time to fix before publishing details.
